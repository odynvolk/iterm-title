# iterm-title
> Super small module for setting iTerm window and tab titles. 

## Install

Install with [npm](https://www.npmjs.com/)

```sh
$ npm install iterm-title --save
```

## Usage

```js

const itermTitle = require("iterm-title");

itermTitle("Some title you want");
````
